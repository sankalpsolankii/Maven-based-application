# SIMPLE-MAVEN-APPLICATION

This is a simple Java Maven application that calculates Student Attendance Analyser

## Features

- Calculates Student Attendance:
  - Attendance % Calculation
  - Eligibility for Exam(>75%)
  - Warning Status.

## Usage

Run the application to calculate the attendance.

```bash
mvn compile exec:java -Dexec.mainClass="com.student.App"
```
